Restores and enforces the bundled jQuery in the admin.

== Description ==

Restores and enforces the bundled jQuery in the admin if your theme registers a newer version which may not be compatible with WordPress yet.

You can try this plugin [if your menus or widgets screens broke](http://wpdevel.wordpress.com/2011/02/10/if-your-menus-or-widgets-screens-broke/).

== Installation ==

1. Upload `restore-bundled-jquery` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
